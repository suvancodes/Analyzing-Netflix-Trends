# 🎬 Netflix Data Analysis Project

## 📌 Overview

This project focuses on analyzing Netflix's dataset to uncover insights about content distribution, trends over time, and global reach. The analysis is performed using Python and visualized through Tableau dashboards.

---

## 🚀 Objectives

* Analyze Netflix content trends over the years
* Compare Movies vs TV Shows distribution
* Identify top countries producing Netflix content
* Discover the most frequent directors
* Visualize content growth and patterns

## 📊 Key Insights

* 🎥 Movies dominate Netflix content compared to TV Shows
* 📈 Rapid growth in content after 2015
* 🌍 USA and India are top contributors
* 🎬 Certain directors appear frequently across the dataset
* 📅 Content release trends show strong yearly patterns

## ⚙️ How to Run the Project

1. Clone the repository:

```bash
git clone https://github.com/suvancodes/Analyzing-Netflix-Trends.git
```

2. Navigate to project folder:

```bash
cd Netflix_Data_Analysis
```

3. Open Jupyter Notebook:

```bash
jupyter notebook
```

4. Run `code.ipynb`
